# [Tile Jar](https://github.com/la-jarre-a-son/tile-jar)

A website to generate tilings with SVG that can be animated and exported as PNGs.

## Compatibility

Tile Jar requires the use of the FileSystem API and a compatible web browser.
To enable the (now deprecated) FileSystem API, go to `chrome://flags` or `brave://flags` and enable the `File System Access API`.

## Run locally

This project is built with Vite

```
npm install

npm run dev
 -or-
npm run build
```
